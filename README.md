# Java-Test
后台管理系统（练习小项目）
## 项目环境
* JDK：jdk1.8.0_45
* TOMCAT:apache-tomcat-8.5.16
* MAVEN:apache-maven-3.3.3
* MySQL:mysql-5.7.28-winx64
* 框架：SSM
* 前端框架：BootStrap
* 后端框架：EasyUI
* 开发工具：IntelliJ_IDEA
> 工具下载地址：<https://github.com/Hi-world-DF/JD_tools>
### 目前项目进度：
#### 已完成功能--->
* 实现登陆验证
* 实现数据库用户列表的查询
* 实现列表分页功能
* 实现用户添加功能
*****
#### 待完成功能--->
* 实现用户删除功能
* 实现用户编辑功能
* 根据条件查询。。。
